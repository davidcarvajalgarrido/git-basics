# Lección 3 — Ejercicios

## Ejercicio 1: Tu primer repo y commit

1. Crea un directorio, inicialízalo con `git init`.
2. Crea `README.md`, añádelo y confirma con mensaje claro.
3. Muestra el historial en una línea.

## Ejercicio 2: Confirma con intención

1. Crea dos archivos con cambios independientes.
2. Usa `git add -p` para incluir solo una parte.
3. Confirma y revisa el diff del commit.

## Ejercicio 3: Corrige el último commit

1. Edita un archivo ya confirmado.
2. Usa `git commit --amend` para actualizar el mensaje o incluir el cambio.

## Ejercicio 4: Deshacer cambios de trabajo

1. Modifica un archivo y luego descarta los cambios con `git restore`.
2. Retira un archivo del staging con `git restore --staged`.
